# Adobe Experience Manager 적응형 Forms에서 텍스트 입력 유효성 검사 패턴 활성화 및 사용

## 개요

Adobe Experience Manager(Adobe Experience Manager) Forms에서 텍스트 입력 구성 요소에 대한 특정 유효성 검사 형식(예: 주민등록번호, 이메일 주소 또는 우편번호)은 템플릿 정책 수준에서 제어됩니다. 이 안내서에서는 다음 방법을 설명합니다.

1. 템플릿 편집기에 액세스하고 유효성 검사 패턴을 노출하도록 텍스트 입력 구성 요소 정책을 구성합니다.
2. 적응형 양식을 만들고 이러한 유효성 검사 패턴을 텍스트 입력 구성 요소에 적용합니다.

## 사전 요구 사항

- Forms이 활성화된 Adobe Experience Manager 인스턴스에 대한 액세스.
- 템플릿을 편집할 수 있는 권한입니다(일반적으로 `template-authors` 그룹의 일부).
- 적응형 Forms을 만들고 편집할 수 있는 권한입니다.

## 1부: 템플릿에서 유효성 검사 패턴 사용

>[!VIDEO](https://video.tv.adobe.com/v/3480390/)

### 1단계: 템플릿 콘솔 액세스

1. Adobe Experience Manager 홈 화면에서 왼쪽 사이드바의 **도구**(망치 아이콘)를 클릭합니다.
2. **일반 > 템플릿**(으)로 이동합니다.
3. 프로젝트 템플릿이 들어 있는 폴더를 선택합니다(예: 핵심 구성 요소 예).
4. 수정하려는 특정 템플릿(예: AF Blank)을 찾아 선택합니다.
5. **편집**&#x200B;을 클릭하거나 템플릿을 직접 엽니다.

### 2단계: 구성 요소 구조 액세스

1. **구조** 보기에 있는지 확인합니다(필요한 경우 오른쪽 상단의 모드 드롭다운에서 &quot;구조&quot;를 선택).
2. 텍스트 입력 구성 요소가 허용되는 기본 **레이아웃 컨테이너** 또는 특정 컨테이너를 찾습니다.
3. 컨테이너를 클릭하여 허용된 구성 요소 목록을 표시합니다.
4. 목록에서 **적응형 양식 텍스트 상자** 구성 요소를 찾습니다.
5. 구성 요소 이름 옆에 있는 **정책** 아이콘(파일/용지함 기호로 표시)을 클릭합니다.

### 3단계: 텍스트 입력 구성 요소 정책 구성

1. 정책 편집기에서 창의 오른쪽에 있는 **형식** 탭을 클릭합니다.
2. 사용 가능한 유효성 검사 패턴 목록이 표시됩니다. 활성화할 형식의 확인란을 선택합니다.
   - 전화 번호
   - 사회 보장 번호
   - 이메일 영숫자
   - 우편번호

### 4단계: 정책 정의 및 저장

1. **정책** 탭으로 다시 전환하거나 왼쪽 설정을 확인하십시오.
2. **정책 제목** 필드에 명확한 이름(예: `new-textinput-default-policy`)을 입력하십시오.
3. 오른쪽 상단의 **완료**(확인 표시) 아이콘을 클릭하여 변경 내용을 저장합니다.

## 2부: 양식 만들기 및 유효성 검사 패턴 사용

이제 템플릿에서 유효성 검사 패턴을 활성화했으므로 적응형 양식을 만들어 텍스트 입력 구성 요소에 적용할 수 있습니다.

>[!VIDEO](https://video.tv.adobe.com/v/3480391/)

### 1단계: 적응형 양식 만들기

1. **Forms 및 문서** 인터페이스로 이동합니다.
2. 오른쪽 상단의 **만들기** 단추를 클릭합니다.
3. 드롭다운 옵션에서 **적응형 양식**&#x200B;을(를) 선택합니다.
4. 유효성 검사 패턴을 사용할 수 있도록 설정한 템플릿(예: **Blank**)을 선택하고 **다음**&#x200B;을 클릭합니다.
5. 속성 추가 화면에서 다음을 수행합니다.
   - 양식에 대한 **제목**&#x200B;을 입력하십시오(예: &quot;testPolicy&quot;). 제목을 기반으로 **이름** 필드가 자동으로 채워집니다.
   - 올바른 **테마 클라이언트 라이브러리**&#x200B;를 선택했는지 확인하십시오(예: `adaptiveform.theme.canvas3`).
6. **만들기**&#x200B;를 클릭합니다. 성공 메시지가 나타납니다.
7. 편집기에서 양식을 열려면 **편집**&#x200B;을 클릭하세요.

### 2단계: 텍스트 입력 구성 요소 추가

1. 양식 편집기의 왼쪽 사이드바에서 **구성 요소** 브라우저를 찾습니다.
2. 검색 창을 사용하여 **적응형 양식 텍스트 상자** 구성 요소를 찾으십시오. &quot;텍스트&quot;를 입력하면 목록이 필터링됩니다.
3. **적응형 양식 텍스트 상자** 구성 요소를 클릭하여 기본 캔버스 영역으로 끌어서 놓습니다.

### 3단계: 구성 요소 서식 구성

1. 새로 추가된 **텍스트 입력** 구성 요소를 클릭하여 선택합니다.
2. 구성 요소 위에 표시되는 도구 모음에서 **구성** 아이콘(렌치로 표시됨)을 클릭합니다.
3. 속성 대화 상자에서 **형식** 탭으로 이동합니다.
4. **Type** 드롭다운 메뉴를 찾아 선택 항목을 **전화 번호**(또는 다른 활성화된 형식)으로 변경합니다.

### 4단계: 데이터 유효성 검사 구성

1. 구성 요소 구성 대화 상자에서 **유효성 검사** 탭으로 이동합니다.
2. **유효성 검사 패턴** 섹션을 찾습니다.
3. **Type** 드롭다운 메뉴를 클릭합니다.
4. 유효성 검사 패턴 목록에서 **전화 번호**&#x200B;을(를) 선택하십시오. 이렇게 하면 사용자가 올바른 전화 번호 형식과 일치하지 않는 텍스트를 입력하는 경우 양식에 오류가 발생합니다.
5. 대화 상자의 오른쪽 상단에 있는 **완료**(확인 표시) 아이콘을 클릭하여 변경 내용을 저장합니다.

## 인증

유효성 검사 패턴이 올바르게 작동하는지 확인하려면 다음을 수행하십시오.

1. 편집기에서 양식을 미리 봅니다.
2. **텍스트 입력** 구성 요소에 테스트 데이터를 입력하십시오.
3. 다음을 확인합니다.
   - 활성화된 유효성 검사 패턴은 구성 요소의 형식 및 유효성 검사 탭에 나타납니다.
   - 잘못된 입력이 적절한 오류 메시지를 트리거합니다.
   - 유효한 입력은 오류 없이 허용됩니다.

## 문제 해결

- **문제:** 유효성 검사 패턴이 양식 구성 요소에 표시되지 않습니다.
   - **해결 방법:** 템플릿 정책이 올바르게 저장되었는지, 양식을 만들 때 올바른 템플릿을 사용하고 있는지 확인하십시오.

- **문제:** 템플릿 편집기에 액세스할 수 없습니다.
   - **해결 방법:** 템플릿을 편집하는 데 필요한 권한(`template-authors` 그룹의 구성원)이 있는지 확인하십시오.

- **문제:** 텍스트 입력 구성 요소가 입력 유효성을 올바르게 검사하지 않습니다.
   - **해결 방법:** 구성 요소의 [유효성 검사] 탭에서 유효성 검사 패턴 설정을 다시 확인하고 올바른 패턴이 선택되어 있는지 확인하십시오.

## 다음 단계

Adobe Experience Manager 적응형 Forms에서 텍스트 입력 유효성 검사 패턴을 활성화하고 사용한 후:

- 다른 데이터 유형에 대한 추가 유효성 검사 패턴을 구성합니다.
- 다른 구성 요소 정책을 탐색하여 양식 동작을 사용자 지정합니다.
- 양식 제출 처리 및 데이터 처리 논리를 설정합니다.
