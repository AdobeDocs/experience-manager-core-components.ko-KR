# 텍스트 구성 요소{#text-component}

코어 구성 요소 텍스트 구성 요소는 즉석 편집 기능을 하는 리치 텍스트 편집 및 구성 구성 요소입니다.

## 사용량 {#usage}

텍스트 구성 요소는 간소화된 인라인 편집기와 전체 화면 포맷으로 쉽게 텍스트를 편집할 수 있도록 해주는 강력한 리치 텍스트 편집기를 제공합니다.

[편집 대화 상자](#edit-dialog)에서는 전체 화면 편집 대화 상자에서 전체 기능을 사용할 수 있는 제한된 옵션과 함께 인라인 편집 기능을 사용할 수 있습니다. [디자인 대화 상자](#design-dialog)를 사용하면 제목, 특수 문자 및 단락 스타일과 같은 텍스트 형식 옵션을 컨텐츠 작성자의 템플릿에 대해 구성할 수 있습니다.

## 버전 및 호환성 {#version-and-compatibility}

텍스트 구성 요소의 현재 버전은 v2이며, 2018년 1월에 핵심 구성 요소 릴리스 2.0.0에서 도입되었으며, 이 문서에 설명되어 있습니다.

다음 표에서는 구성 요소의 지원되는 모든 버전, 구성 요소의 버전이 호환되는 AEM 버전 및 이전 버전에 대한 설명서 링크에 대해 자세히 설명합니다.

| 구성 요소 버전 | AEM 6.4 | AEM 6.5 | AEM as a Cloud Service |
|---|---|---|---|
| v2 | 호환 가능 | 호환 가능 | 호환 가능 |
| [v1](v1/text-v1.md) | 호환 가능 | 호환 가능 | - |

핵심 구성 요소 버전 및 릴리스에 대한 자세한 내용은 [핵심 구성 요소 버전](/help/versions.md) 문서를 참조하십시오.

## 텍스트 구성 요소 및 리치 텍스트 편집기 {#the-text-component-and-the-rich-text-editor}

코어 구성 요소 텍스트 구성 요소는 AEM 리치 텍스트 편집기(RTE)를 활용합니다. RTE는 컨텐츠 작성자가 텍스트 컨텐츠를 편집할 수 있는 다양한 기능을 제공합니다. RTE는 구성 면에서 매우 유연하며 다양한 옵션을 제공합니다. RTE 구성 방법에 대한 자세한 내용은 [리치 텍스트 편집기 구성](https://docs.adobe.com/content/help/ko/experience-manager-cloud-service/implementing/configuring-and-extending/rich-text-editor.html) 및 [리치 텍스트 편집기 플러그인 구성](https://docs.adobe.com/content/help/ko/experience-manager-cloud-service/implementing/configuring-and-extending/configure-rich-text-editor-plug-ins.html)에서 확인할 수 있습니다.

이 문서의 나머지 부분에서는 기본 제공 RTE 구성과 함께 코어 구성 요소 텍스트 구성 요소의 표준 구성을 보여줍니다.

>[!NOTE]
>
>RTE](https://docs.adobe.com/content/help/en/experience-manager-cloud-service/implementing/configuring-and-extending/configure-rich-text-editor-plug-ins.html)의 [UI 구성에 의해 활성화된 옵션만 텍스트 구성 요소에서 사용할 수 있습니다.

## 편집 대화 상자 {#edit-dialog}

편집 대화 상자는 사용자가 텍스트를 작성할 것으로 예상되는 표준 서식 있는 텍스트 서식 도구를 제공합니다.

![텍스트 구성 요소의 편집 대화 상자](/help/assets/text-edit.png)

### 굵게

![굵게 아이콘](/help/assets/text-bold.png)

선택한 텍스트에 굵은 서식을 적용하거나 커서 뒤에 입력한 텍스트를 굵게 서식을 지정하는 데 사용됩니다.

**Ctrl+** Bcan을 키보드 단축키로 사용할 수 있습니다.

### 기울임체

![기울임체 아이콘](/help/assets/text-italic.png)

선택한 텍스트에 기울임꼴 서식을 적용하거나 커서 뒤에 입력한 텍스트를 기울임꼴로 만드는 데 사용됩니다.

**Ctrl+** K를 키보드 단축키로 사용할 수 있습니다.

### 밑줄

![밑줄 아이콘](/help/assets/text-underline.png)

선택한 텍스트에 밑줄이 그어진 서식을 적용하거나 커서 뒤에 입력한 텍스트에 밑줄을 긋는 데 사용됩니다.

**Ctrl+** Uca를 키보드 단축키로 사용할 수 있습니다.

### 아래 첨자

![하위 스크립트 아이콘](/help/assets/text-subscript.png)

커서 뒤에 입력한 선택한 텍스트나 텍스트의 서식을 하위 스크립트로 지정하는 데 사용됩니다.

### 위 첨자

![위 첨자 아이콘](/help/assets/text-superscript.png)

커서 뒤에 입력한 텍스트 또는 텍스트를 위 첨자로 서식을 지정하는 데 사용됩니다.

### 텍스트로 붙여넣기

![텍스트로 붙여넣기 아이콘](/help/assets/text-paste-text.png)

복사한 텍스트를 서식 없이 일반 텍스트로 붙여넣습니다.

이 옵션을 선택하면 텍스트를 텍스트에 삽입하기 전에 미리 보기로 서식이 없는 일반 텍스트로 붙여넣을 수 있는 창이 열립니다. 확인 표시를 탭하거나 클릭하여 승인하고, x를 탭하거나 클릭하여 취소합니다.

![텍스트 예로 붙여넣기](/help/assets/text-paste-text-example.png)

### Word에서 붙여넣기

![Word 아이콘에서 붙여넣기](/help/assets/text-paste-word.png)

이 옵션을 선택하면 텍스트에 삽입되기 전에 텍스트 서식을 미리 보기로 유지하여 붙여넣을 수 있는 창이 열립니다. 확인 표시를 탭하거나 클릭하여 승인하고, x를 탭하거나 클릭하여 취소합니다.

![Word에서 붙여넣기 예](/help/assets/text-paste-word-example.png)

### 하이퍼링크

![하이퍼링크 아이콘](/help/assets/text-hyperlink.png)

이 옵션을 사용하여 선택한 텍스트를 하이퍼링크로 변환하거나 이미 정의된 링크를 수정합니다. 이 옵션은 텍스트를 이미 선택하고 링크를 설정할 수 있는 추가 옵션이 있는 창을 여는 경우에만 활성화됩니다.

![하이퍼링크 예](/help/assets/text-hyperlink-example.png)

* 경로를 입력합니다
   * [선택 열기] 대화 상자를 사용하여 AEM에서 경로를 선택합니다
   * 링크가 AEM 내에 없으면 절대 URL을 입력합니다
      * 절대 경로가 아닌 경로는 AEM을 기준으로 해석됩니다
* 링크에 대한 대체 설명 텍스트를 입력합니다
* 링크 동작 선택
   * 타겟
   * 동일한 탭
   * 새 탭
   * 상위 프레임
   * 상위 프레임

   링크를 적용하려면 확인 표시를 탭하거나 클릭하고 취소할 x를 클릭합니다.

### 연결 해제

![연결 해제 아이콘](/help/assets/text-unlink.png)

선택한 텍스트에 이미 적용된 링크를 제거하려면 이 옵션을 사용합니다. 이 옵션은 링크가 이미 선택된 경우에만 활성화됩니다.

### 찾기

![찾기 아이콘](/help/assets/text-find.png)

텍스트를 검색하여 지정된 텍스트 문자열을 검색합니다. 이 옵션을 선택하면 검색 옵션을 지정할 창이 열립니다.

![찾기 예](/help/assets/text-find-example.png)

검색할 텍스트를 입력하고 **찾기**를 클릭하거나 클릭하여 검색을 시작합니다. 취소하려면 x 를 탭하거나 클릭합니다.
사례에 따라 정확히 일치시키려면 검색을 시작하기 전에 **대/소문자 일치** 옵션을 선택합니다.
일치하는 항목이 있으면 강조 표시되고 검색 대화 상자가 흐리게 표시됩니다. 흐리게 표시된 대화 상자에서 **찾기** 단추를 다시 탭하거나 클릭하여 다음 항목을 검색합니다.

![찾기 예 찾음](/help/assets/text-find-example-found.png)

추가 항목이 없으면 메시지가 표시되고 텍스트 시작부터 검색이 다시 시작됩니다.

![예제를 더 이상 찾을 수 없음](/help/assets/text-find-example-found-end.png)

### 바꾸기

![바꾸기 아이콘](/help/assets/text-replace.png)

텍스트를 검색하여 지정된 텍스트 문자열을 찾은 후 일치 항목을 다른 문자열로 대체하려면 이 옵션을 사용합니다. 이 옵션을 선택하면 검색 및 바꾸기 옵션을 지정하는 창이 열립니다.

![바꾸기 예](/help/assets/text-replace-example.png)

검색할 텍스트와 바꿀 텍스트를 입력합니다.

* **찾기**&#x200B;를 탭하거나 클릭하여 검색을 시작합니다. 취소하려면 x 를 클릭하거나 탭합니다.
* 사례에 따라 정확히 일치시키려면 검색을 시작하기 전에 **대/소문자 일치** 옵션을 선택합니다.
* **모두 바꾸기**&#x200B;를 선택하여 텍스트 발생 항목을 한 번에 모두 바꿉니다.

일치하는 항목이 있으면 강조 표시되고 검색 대화 상자가 흐리게 표시됩니다. 희미하게 대화 상자에서 **찾기** 단추를 다시 클릭하여 다음 항목을 검색하거나 **바꾸기** 단추를 선택하여 강조 표시된 일치하는 텍스트를 바꿉니다. **Replace** 단추는 일치하는 경우에만 활성화됩니다.

찾기를 클릭하면 찾기 및 바꾸기 대화 상자가 투명해지고 바꾸기를 클릭하면 불투명해집니다. 이를 통해 작성자는 작성자가 대체할 텍스트를 검토할 수 있습니다.

>[!NOTE]
>
>바꾸기 기능을 사용할 때 바꿀 바꾸기 문자열을 찾기 문자열과 동시에 입력해야 합니다. 그러나 문자열을 대체하기 전에 찾기 를 클릭하여 검색할 수 있습니다. 찾기를 클릭한 후 바꾸기 문자열을 입력하면 검색이 텍스트 시작 부분으로 재설정됩니다.

### 표

![표 아이콘](/help/assets/text-table.png)

텍스트에 표를 삽입하는 데 사용됩니다. 이 옵션을 선택하면 테이블의 세부 정보를 지정할 수 있는 창이 열립니다.

![표 예](/help/assets/text-table-example.png)

* **열**  - 테이블의 열 수(필수)
* **행**  - 테이블의 행 수(필수)
* **너비**  - 표의 폭입니다
* **높이**  - 표의 높이입니다
* **셀 패딩**  - 셀 콘텐츠 주위의 공간
* **셀 간격**  - 셀 사이의 간격
* **테두리**  - 표의 테두리 선 두께
   * 테이블 헤더에 대해
      * 첫 번째 행을 사용해야 합니다
      * 첫 번째 열을 사용해야 합니다
      * 첫 번째 행과 첫 번째 열을 사용해야 합니다
      * 또는 헤더를 사용하지 않아야 합니다.
* **캡션**  - 표의 캡션

### 특수 문자 {#special-characters}

![특수 문자 아이콘](/help/assets/text-special-characters.png)

텍스트에 특수 문자를 삽입하는 데 사용됩니다. 이 옵션을 선택하면 사용 가능한 문자가 표시되는 창이 열립니다.

![특수 문자 예](/help/assets/text-special-characters-example.png)

원하는 문자를 탭하거나 클릭하여 커서 뒤에 텍스트에 삽입합니다. 여러 문자를 삽입할 수 있습니다. x 를 탭하거나 클릭하여 선택 창을 닫습니다.

### 소스 편집

![소스 편집 아이콘](/help/assets/text-source.png)

텍스트의 HTML 소스를 보고 수정하는 데 사용됩니다.

**소스 편집** 아이콘을 탭하거나 클릭하여 서식이 지정된 보기에서 텍스트 컨텐츠를 변경하여 원시 HTML을 봅니다. 이 모드에서는 다른 모든 서식 옵션이 비활성화됩니다. **소스 편집** 아이콘을 다시 탭하거나 클릭하여 형식이 지정된 보기로 돌아갑니다.

>[!CAUTION]
>
>원시 HTML에 액세스할 수 있는 모든 경우처럼 **소스 편집** 옵션을 사용할 때에는 주의해야 합니다.
>
>**소스 편집**&#x200B;을 통해 입력한 HTML이 XSS 위험에 대해 검색되고 삽입된 모든 스크립트는 제거되고 결과 페이지에 나타나지 않습니다. 그러나 **소스 편집**&#x200B;에 입력한 HTML의 형식이 잘못되어 페이지에 대한 템플릿이 중단되거나 이로 인해 예기치 않은 서식이 지정되거나 결과 페이지를 사용할 수 없게 될 수 있습니다.

>[!NOTE]
>
>**소스 편집**&#x200B;을 통해 입력한 HTML이 XSS 위험 및 스크립트를 스캔하고 찾은 스크립트를 자동으로 제거하므로, 지속되는 실제 컨텐츠는 **소스 편집**&#x200B;에 입력한 내용과 다를 수 있습니다. 이러한 이유로 **소스 편집**&#x200B;을 사용하여 변경한 내용을 저장하려면 먼저 **소스 편집**&#x200B;을 종료하여 일반 편집기에서 텍스트를 확인해야 합니다.

### 단락 형식

선택한 텍스트나 커서 뒤에 삽입된 텍스트에 단락 서식을 적용하는 데 사용됩니다. 이 옵션을 선택하면 단락 형식이 선택된 드롭다운이 열립니다.

![단락 서식 예](/help/assets/text-paragraph-example.png)

### 인라인 편집 {#in-line-editing}

텍스트 구성 요소는 인라인 편집도 가능하지만, 공간 제한으로 인해 모든 서식 옵션을 인라인 사용할 수는 없습니다. 모든 옵션을 보려면 전체 화면 모드로 전환합니다.

![인라인 편집 예](/help/assets/text-edit-inline-example.png)

### 설정 및 ID {#setting-id}

이 옵션을 사용하면 HTML과 [데이터 레이어](/help/developing/data-layer/overview.md)에서 구성 요소의 고유 식별자를 제어할 수 있습니다.

* 비워 두면 고유 ID가 자동으로 생성되며 결과 페이지를 검사하여 찾을 수 있습니다.
* ID가 지정된 경우 ID가 고유한지 확인하는 것은 작성자의 책임입니다.
* ID를 변경하면 CSS, JS 및 데이터 레이어 추적에 영향을 줄 수 있습니다.

## 디자인 대화 상자 {#design-dialog}

디자인 대화 상자에서는 템플릿 작성자가 사용할 수 있는 텍스트 형식 옵션을 정의할 수 있습니다.

### Plugins 탭 {#plugins-tab}

플러그인 탭은 컨텐츠 작성자가 사용할 수 있는 다양한 텍스트 형식 옵션을 활성화하고 비활성화하는 데 사용됩니다.

### 기능 {#features}

![디자인 대화 상자 기능](/help/assets/text-design-features.png)

구성 요소에 대해 다음 기능을 활성화하거나 비활성화할 수 있습니다.

* 일반 텍스트 붙여넣기
* 이전
* 찾기 및 바꾸기
* 맞춤법 검사
* 삽입된 이미지 수정 옵션
* HTML 소스 편집

### 서식 {#formatting}

![디자인 대화 상자 서식](/help/assets/text-design-formatting.png)

구성 요소에 대해 다음 서식 옵션을 활성화하거나 비활성화할 수 있습니다.

* 표
* 목록(글머리 기호, 번호, 들여쓰기, 내어쓰기)
* 정렬(왼쪽, 오른쪽, 가운데)
* 굵게, 기울임체, 밑줄
* 연결(및 연결 해제)
* 아래/위 첨자

### 단락 스타일 {#paragraph-styles}

![디자인 대화 상자 단락 스타일](/help/assets/text-design-paragraph.png)

구성 요소에 대해 단락 스타일을 활성화하거나 비활성화할 수 있습니다. 활성화되면 허용되는 형식을 정의할 수 있습니다.

* **추가** 단추를 탭하거나 클릭하여 새 스타일을 삽입합니다.
* 편집 대화 상자에 표시할 스타일 코드와 설명을 입력합니다.
* 스타일을 제거하려면 **삭제** 단추를 탭하거나 클릭합니다.
* 서식 순서를 재정렬하려면 핸들을 탭하거나 클릭하고 드래그합니다.

### 특수 문자 {#configuring-special-characters}

![디자인 대화 상자 특수 문자](/help/assets/text-design-special-characters.png)

구성 요소에 대해 특수 문자를 삽입하는 옵션을 활성화하거나 비활성화할 수 있습니다. 활성화되면 허용되는 문자를 정의할 수 있습니다.

* **추가** 단추를 탭하거나 클릭하여 새 문자를 삽입합니다.
* 문자의 HTML 코드와 편집 대화 상자에 표시될 설명을 입력합니다.
* 문자를 제거하려면 **삭제** 단추를 탭하거나 클릭합니다.
* 문자의 순서를 재정렬하려면 을 탭하거나 클릭하고 핸들을 드래그합니다.
